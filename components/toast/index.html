toast组件

实例代码

var toast = new jam.cUIToast({
    title: '带蒙版，点击蒙版不关闭，3秒关闭',
    disableScroll: true,
    sleep: 3,
    needMaskHide: false,
    // prefix: 'xxxx-',
    delayCallback: function(){
        console.log('关闭')
    }
});
toast.show();

API

方法名 说明 参数名 参数描述

title 提示文字 String '您好！'
sleep 持续时间 number
