获取服务器时间，只需获取一次服务器时间，之后通过计算服务器跟客户端跟服务器时间差来获得服务器时间。

实例代码

var __SERVERDATE__ = {
    server: new Date(Date.parse('2018-02-09 12:03:49'.replace(/-/g, '/'))),
    local: new Date()
}
jam.servertime.getServerDate(function(d){
    var _serverTime = d;
})

API

方法名 说明 参数名 参数描述
getServerDate 获取服务端时间 callback 需要页面上赋值 __SERVERDATE__服务器时间
