  • Validate

平时开发中用到的一些常用的验证方法集合。

实例代码

jam.validate.isChinese('123123') // false
jam.validate.isChinese('hello') // false
jam.validate.isChinese('哈喽') // true
jam.validate.isMobile('13812341234') // true 手机号码
jam.validate.isCellPhone('013812341234') // true 手机号码前面可+0
jam.validate.isPhone('021-1231423') // true 座机号码
jam.validate.isIdCard('430404196710021020') // true 身份证号码

API

方法名 说明 参数名 参数描述
isEmail 验证邮箱 String 需要验证的邮箱
isPassword 验证密码强度，6-20位大小写字母与数字组成 String 需要验证的密码
isMobile 验证手机号 String 需要验证的手机号
isChinese 验证中文字 String 需要验证的字符串
isEnglish 验证英文字母 String 需要验证的字符串
isZip 验证邮编 String 需要验证的邮编
isDate 验证日期格式 String 需要验证的日期
isNum 验证数字 String 需要验证的字符串
isCellPhone 验证座机号码格式 String 需要验证的座机号码
isIDCardNo 验证ID String 需要验证的ID
isEnglishAndSpace 验证英文和英文空格 String 需要验证的字符串
isTraditional 验证繁体字 String 需要验证的字符串
isIdCard 验证身份证 String 需要验证的身份证号码 18位身份证号码
isRightVerifycode 验证验证码 String 4-30位小写字母以及数字
isAllowSetTradingPass 验证交易密码 String 禁止连续相同数字或者字母
isQq 验证QQ号码 String 需要验证的字符串
isPhone 验证座机号码 String 需要座机号码 带区号
isUrl 验证网站地址 String 需要验证的字符串
isIP 验证IP地址 String 需要验证的字符串
isEmptyObject 验证空对象 Json
