获取服务器时间，只需获取一次服务器时间，之后通过计算服务器跟客户端跟服务器时间差来获得服务器时间。

实例代码

jam.date.format(new Date,'Y-m-d H:i:s') // 输出 2018-02-09 12:03:49 格式
jam.date.getIntervalDay('2018-02-09 12:03:49', '2017-02-09 12:03:49') // -365
// ...

API

方法名 说明 参数名 参数描述
diffMonth 计算两个时间的相隔月份数 d1, d2 两个时间的时间戳
format 日期类型格式化为指定字符串 obj, str 参考 http://php.net/manual/zh/function.date.php
getHM 返回HH：MM格式 timeStr
getIntervalDay 返回两个日期相差的天数 ds1, ds2
m2H 将分钟转还为小时字符串，如90输出为1小时30分钟 min
parse 将字符串转换为CDate对象 str
weekday 获取周几，d为日期C.parse(d); date Returns 周日，周一...周六
